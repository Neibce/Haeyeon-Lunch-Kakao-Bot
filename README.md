# Hayeon-Lunch-Bot (해연중학교 급식 알리미 봇)
- 카카오톡 플러스친구 '해연중학교 급식 알리미'의 급식 메뉴 자동 응답 봇 코드
- 급식 메뉴 알고 싶은 날의 버튼 클릭 시 해당 날짜의 급식 메뉴 반환
- 카카오톡 플러스친구/옐로우아이디 자동 응답 API 기반
- Apache, PHP7, MySQL
- 2017년 4분기 ~ 2018년 1분기
